GeoComponents

GeoComponents is a collection of web components and custom elements used to support various geoinformation projects. The geocomponents.org acts as a reliable host for the components to encourage sharing and reuse.

Features

Fast.

Static files served from Google Cloud.

Minimalistic.

Simple custome elements built on pure JS (ECAMScript 6)

For every project.

Served over https and open for inclusion in any domain.
