This is a test to build up a simple web component for the Gplates web service. By building this as a component it is far easier for people to simply include a call to the service and display a map.

Next step will be to call the point reprojection service to place points on the map associated with features of interest. For IODP this will be a cruise path projected back to the known basement age of the expedition.

The code required to include this into a page by a author is:

Head
<script src="/components/leaflet-element/bower_components/webcomponentsjs/webcomponents-lite.js"></script>
<link rel="import" href="/components/ocdgplates-element/ocdgplates-element.html">
Body
<ocdgplates-element age="140"></ocdgplates-element>
