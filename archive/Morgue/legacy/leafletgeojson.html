This component and the other leaflet component on this site use the well crafted Leaflet package web component ( https://leaflet-extras.github.io/leaflet-map/demo.html). This is not developed by GeoLink but is used as a component and then data bound to the Polymer ( https://www.polymer-project.org/1.0/) Iron-Ajax component. When connected to services that expose the GeoLink data holdings a simple approach to data visualization is the result. This approach is also easy for others to incorporate into their work.
